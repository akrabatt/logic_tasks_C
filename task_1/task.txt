Задача: Напиши функцию на языке C, которая принимает массив целых чисел и его размер, и переставляет все нули в конец массива, сохраняя относительный порядок ненулевых элементов. Решение должно быть in-place (без использования дополнительных массивов).

Пример:
Вход: [0, 1, 0, 3, 12]
Выход: [1, 3, 12, 0, 0]

Подсказка:
Попробуй использовать два «указателя» (индекса) для эффективного перемещения элементов.

Когда напишешь решение, проверь его на разных тестах:

[1, 0, 2, 0, 4] → [1, 2, 4, 0, 0]

[0, 0, 0] → [0, 0, 0]

[5, 3, 0, 2] → [5, 3, 2, 0]

Какова временная сложность твоего алгоритма?

Можно ли сделать это за один проход?

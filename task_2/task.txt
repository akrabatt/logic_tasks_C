Задача: Напиши функцию на языке C, которая принимает массив целых чисел и его размер, и возвращает количество положительных чисел в массиве.

Примеры:

Вход: [5, -2, 0, 10, -7] → Выход: 3 (положительные: 5, 10)

Вход: [-1, -5, 0] → Выход: 0

Вход: [8, 12, 3] → Выход: 3

Условия:

Массив может содержать отрицательные числа, нули и положительные.

Ноль не считается положительным числом.

Функция должна называться count_positive.

Подсказка:
Используй цикл для прохода по массиву и условие для проверки arr[i] > 0.

Проверь себя:

Как обработать пустой массив?

Что, если все элементы нулевые?

Учитывается ли 0 как положительное? (Нет!)

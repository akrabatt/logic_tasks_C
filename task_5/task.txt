Отлично! Давай перейдем к задаче на реверс массива in-place. Это поможет закрепить работу с индексами, циклами и манипуляцией элементами массива.

Задача:
Напиши функцию на C, которая принимает массив целых чисел и его размер, и переворачивает элементы массива in-place (без использования дополнительного массива).

Примеры:

Вход: [1, 2, 3, 4, 5] → Выход: [5, 4, 3, 2, 1]

Вход: [9, 0, -3] → Выход: [-3, 0, 9]

Вход: [] → Выход: []

Условия:

Функция должна называться reverse_array.

Изменяй исходный массив, а не создавай новый.

Используй только циклы и временные переменные.

Подсказка:
Используй два индекса:

start (начало массива, начинает с 0)

end (конец массива, начинает с size - 1)

На каждой итерации цикла меняй местами arr[start] и arr[end], затем увеличивай start и уменьшай end, пока start < end.

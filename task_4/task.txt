Задача:
Напиши программу на C, которая находит минимальный и максимальный элементы в массиве целых чисел. Результат должен возвращаться в виде структуры.

Требования:

Создай структуру MinMax с полями:

int min

int max

Напиши функцию find_min_max, которая принимает массив и его размер, и возвращает структуру MinMax.

Если массив пустой, верни структуру с min = 0 и max = 0.

Решение должно быть эффективным (проход по массиву за один цикл).

Примеры:

Вход: [5, -2, 0, 10, -7] → Выход: min = -7, max = 10

Вход: [8] → Выход: min = 8, max = 8

Вход: [] → Выход: min = 0, max = 0

Подсказки:
Инициализируй min и max первым элементом массива (если массив не пустой).

В цикле сравнивай текущий элемент с min и max, обновляя их при необходимости.

Для пустого массива сразу верни min = 0, max = 0.
